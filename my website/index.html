Projects

Design Of a Video Stream Analytic System By using OpenCV, Kafka,Spark technologies

M.Tech thesis project (Ongoing) • October, 2018 - (cont.)

Kafka producer will produce KEY-FRAMES (by calculating ENTROPY) from CCTV cameras, and will continuously send them to Kafka consumer. The consumer will process them using Spark, for high-speed processing.

Design Of a Voice-Based System by Recognizing Speech Using MFCC

M.Sc thesis project • November 2015 - July 2016

It is an automated speech recognition s/w which will execute various commands to perform basic computer task such as opening task manager, notepad, calculator etc. The ‘voice-commands’ are processed and then Mel Frequency Cepstral Coeﬃcients (MFCC) are calculated to extract the characteristics of training/testing data.

Design Of an Application using JAVA GUI for chatting and direct video calling

B.Sc thesis project • February 2013 - June 2014

The server will connect 2 clients, then video calling will happen directly from host to host, the overhead of server is optimized. The host can multicast/broadcast messages to other online/offline users. Some security is provided here, if the user moves away from the web camera then the client s/w locks itself. It is done by OpenCV.
